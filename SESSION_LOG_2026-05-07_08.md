Продолжение `SESSION_LOG_2026-05-06.md`. Большой кусок экспериментов: Hysteria2, AmneziaWG, NaiveProxy, AnyTLS, Trojan-WS, Shadowsocks 2022, Bunny CDN, **анализ StealthSurf**.

## Ключевой вывод

**На одном IP единичный VPN endpoint в MTS не выживает дольше 30 секунд.** Любой протокол → DPI пропускает первые байты → собирает статистику flow → шейпит трафик до 0 КБ/с.

Решение — **не протокол, а IP/AS**. StealthSurf показал нам это.

## Что мы знаем о MTS DPI (фактический research)

### Mobile MTS (4G/LTE)
- **Whitelist mode** активирован с начала 2026 в большинстве регионов
- Разрешены ~531 IP'ов МТС-операторских + Yandex/VK/Госуслуги/банки
- Всё остальное режется на L3 — даже SYN не доходит до целевого IP
- CDN-обход не работает (Bunny/Akamai/CF IP не в whitelist)
- **Решение**: cascade через RU VPS которого whitelist принимает (Yandex Cloud / VK Cloud — но они тоже флажатся постепенно)

### Home MTS (cable)
- **НЕ whitelist**, но жёсткий ТСПУ
- Пропускает первый TCP SYN/handshake до любого IP
- После 20-40 сек реального VPN трафика (encrypted random pattern) → шейпит до 0 КБ/с
- Ловит **fingerprint**: длинные сессии с равномерным data flow, не похожие на browsing
- Reality сейчас детектится ML (RKN февраль 2026)
- DigitalOcean / Hetzner / OVH IP'ы в priority watch list
- Нишевые провайдеры (Aeza, Tom Gewiese, KZ-хостеры) **проходят**

### Что работает (community knowledge)
- **VLESS Reality на bulletproof IP** (StealthSurf модель)
- **AmneziaWG 2.0 на Amnezia серверах** (где per-user randomized H1-H4)
- **Outline / Shadowsocks 2022 на свежих IP**
- **Cascade через KZ/AM/GE VPS** (~$3/мес)

## Решения / следующие шаги

### Срочно (когда юзер вернётся к этому)

1. **Удалить старые DO дроплеты** (207.154.214.108) — ресурсы освободить
2. **Купить Aeza Frankfurt** ($4/мес) — повторить StealthSurf approach: VLESS Reality, пустой SNI, 8-char shortId, fp=chrome
3. Перенести подписку на новый IP
4. **Бот health-check** добавить — если IP начал шейпиться, пометить как dead, исключить из подписки

### Стратегически (для коммерческого VPN бизнеса на MTS)

1. **3-5 серверов в разных мелких ASN** (KZ, AM, GE, мелкие EU)
2. **Auto-rotation** — бот раз в 30 сек обновляет primary в подписке  
3. **Health-checker с RU монитор-нодой**
4. **Split-tunneling routing** — RU сайты direct, заблокированные через VPN
5. Бюджет: $30-50/мес минимум

### Что НЕ работает (зафиксировать чтоб не повторять)

- Single-IP architecture с DigitalOcean / Hetzner (даже фрешный) — фейл за минуты
- Любая обфускация (Hysteria2 Salamander, AmneziaWG Jc=10, etc) — NS/MTS DPI её ловит
- Bunny CDN или CloudFlare front — не помогает на mobile (whitelist), мешает на home (CDN IP в priority watch)
- IPv6 Reality — у MTS юзеров часто нет IPv6
- Reality с экзотичными SNI (microsoft.com, vk.com, github.com) — DPI ловит regardless
